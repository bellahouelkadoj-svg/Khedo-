# 🏠 Facade AI Designer

App web pour transformer votre façade avec AI — Upload une photo, AI analyse, redesign en 3D.

## 🌐 Live Demo

**[Voir l'app en ligne](https://votre-username.github.io/BKhedo)**

## ✨ Fonctionnalités

### 🤖 AI Detection
- Upload photo de votre façade
- AI détecte automatiquement :
  - 🧱 Murs
  - 🪟 Fenêtres
  - 🚪 Portes
  - 🏠 Toit

### 🎨 Redesign
- **9 matériaux** : Brique, Pierre, Bois, Crépi, Béton, Carrelage, Peintures
- **4 styles de fenêtres** : Classique, Moderne, Bois, Aluminium
- **3 styles de portes** : Bois Massif, Métal, Vitée
- Application en temps réel sur la photo

### 📤 Export
- 📷 **Image PNG** — Partagez votre design
- 📐 **Fichier DXF** — Ouvrez dans AutoCAD / Rhino
- 📄 **Document PDF** — Devis complet

### 💰 Estimation de Prix
- Surface murale automatique
- Prix matériaux par m²
- Main d'œuvre incluse
- Total estimé en DH

## 🚀 Déploiement

### GitHub Pages (Gratuit)

1. Upload ces fichiers dans votre repo GitHub
2. Allez dans **Settings** → **Pages**
3. Source : **Deploy from a branch**
4. Branch : **main** / **root**
5. Sauvegarder → L'app sera en ligne en 2 minutes

## 🛠️ Technologies

- HTML5 / CSS3 / JavaScript
- Canvas API (image processing)
- GitHub Pages (hosting)
